# Pipeline LinkedIn – Sprint 1

Objectif de ce sprint : créer une structure simple pour écrire, stocker et publier des posts LinkedIn optimisés.

## Structure actuelle

- `posts/` : contient les posts LinkedIn au format texte, prêts à être copiés-collés dans LinkedIn.
- `README.md` : documentation du projet et des conventions.

## Format des posts

- Fichiers `.txt`
- Contenu : texte brut, emojis, sauts de ligne, hashtags en bas.
- Utilisation : ouvrir le fichier, copier le contenu, coller dans LinkedIn.

Ce dépôt servira ensuite de base pour automatiser la publication (Zapier, Buffer, GitHub Actions).
